# 如何获取微信公众号文章链接

## 方法一：从精选文章页面获取（推荐）

1. 打开微信公众号精选文章页面：
   https://mp.weixin.qq.com/mp/appmsgalbum?action=getalbum&album_id=1417552598718332928&__biz=MzIxMjYyMDA2Nw==#wechat_redirect

2. 在浏览器中，右键点击每篇文章标题，选择"复制链接地址"

3. 文章链接格式类似：
   ```
   https://mp.weixin.qq.com/s/xxxxxxxxxxxxx
   ```

## 方法二：从文章页面获取

1. 打开任意一篇文章
2. 点击右上角的"..."菜单
3. 选择"复制链接"

## 方法三：使用手动辅助工具

运行以下命令，按提示输入文章链接：

```bash
cd blog/migration
py scripts/manual_fetch.py
```

工具会提示您输入：
- 文章链接
- 文章标题（可选）

## 方法四：手动创建文章列表

1. 复制 `data/articles_list_template.json` 为 `data/articles_list.json`
2. 填写所有文章信息：

```json
[
  {
    "title": "文章标题",
    "url": "https://mp.weixin.qq.com/s/文章链接",
    "timestamp": 1597470506
  }
]
```

**如何获取时间戳？**
- 时间戳是文章发布日期的Unix时间戳
- 可以使用在线工具转换：https://tool.lu/timestamp/
- 或者从文章页面源代码中查找

## 下一步

获取文章链接后，可以：
1. 使用 `manual_fetch.py` 逐个输入链接
2. 或手动创建 `articles_list.json` 文件
3. 然后运行后续脚本完成迁移
